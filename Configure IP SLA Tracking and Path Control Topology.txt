pract 1- Aim: Configure IP SLA Tracking and Path Control Topology

Topology Diagram:take three routers r1-r2 with s1/0 ,r2-r3 with s1/1,r3-r1 with s1/2  

****configure routers 

in case if the telnet console in gns3 isnt working(in cmd write - telnet localhost 5000 or 5001 or 5002 for routers r1,r2 ans r3 respectively)

R1#conf t
int s1/0
ip add 209.165.200.1 255.255.255.0
no sh
int s1/2
ip add 209.165.201.1 255.255.255.0
no sh
do sh ip int br
exit

R2#conf t
int s1/0
ip add 209.165.200.2 255.255.255.0
no sh
int s1/1
ip add 209.165.202.1 255.255.255.0
no sh
do sh ip int br
exit

R3#conf t
int s1/1
ip add 209.165.202.2 255.255.255.0
no sh
int s1/2
ip add 209.165.201.1 255.255.255.0
no sh
do sh ip int br
exit


****Configure loopbacks and assign addresses: 
r1-config> 
router eigrp 1
network 209.165.200.0 0.0.0.255
network 209.165.201.0 0.0.0.255
no auto-summary
ip route 192.168.1.0 255.255.255.0 209.165.201.3

r2-config> 
router eigrp 1
network 209.165.200.0 0.0.0.255
network 209.165.202.0 0.0.0.255
no auto-summary
ip route 192.168.1.0 255.255.255.0 209.165.202.2

r3-config> 
ip route 0.0.0.0 0.0.0.0 209.165.201.1
int lo0
ip add 192.168.1.1 255.255.255.0



*****Pinging the router


Pinging from R1 to R3
R1(config)#do ping 192.168.1.1

Pinging from R2 to R3
R2(config)#do ping 192.168.1.1
